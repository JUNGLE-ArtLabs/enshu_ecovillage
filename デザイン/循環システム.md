・バイオジオフィルター
🧪 バイオジオフィルター（Bio-Geofilter）概要

微生物（bio）× 地質素材（geo）で構成された浄化システム。
水・空気・土壌に含まれる汚染物質を、微生物の代謝と鉱物・土壌の吸着/イオン交換/物理ろ過を組み合わせて取り除く。
パーマカルチャー圏では“自然界の浄化プロセスを小型に再現した装置”として扱われる。

—

🌏 1. 目的

水の浄化：有機物、アンモニア、硝酸塩、重金属、農薬等の減少

土壌再生：微生物群集の活性化、有害物質の固定化

空気浄化（希少）

エコビレッジの循環システムの中核にしやすい

生活排水 → バイオジオ → 果樹エリアへ灌漑

養鶏・堆肥 → バイオジオ → 再利用

水と土が静かに呼吸する“自然の腎臓”を人工的に作るイメージ。

—

🔬 2. 基本構造（階層）
① 物理フィルター層

砂

砂利

活性炭
汚れ・浮遊物を止める。

② 地質・鉱物層（Geo layer）

ゼオライト（アンモニア吸着）

バイオチャー

粘土鉱物（イオン交換）
“無言で働く鉱物ワーカー”たちが重金属やアンモニアを固定する。

③ バイオフィルム層（Bio layer）

硝化菌・脱窒菌

有機物分解菌

菌根菌（用途による）
ここが心臓部で、汚染物を別の化学形に代謝し無害化する。

④ 仕上げ植物層（Constructed wetland zone / optional）

ヨシ、パピルス、ガマなど
根圏が微生物の住処＝バイオフィルムを強力に増やす。

—

🧯 3. 主な浄化メカニズム

吸着：鉱物・炭が汚染物をつかむ

イオン交換：ゼオライトや粘土の特性

生物分解：バクテリアが有機物を食べる

変換：アンモニア → 亜硝酸 → 硝酸 → N₂ガスへ

固定化：重金属を鉱物が閉じ込める

自然界の“土壌の化学工場”をミニaturizeした構造。

—

🛠 4. 遠州エコビレッジでの利用可能性

あなたのプロジェクト文脈で読むと、バイオジオフィルターは特に有効👇

① 生活排水の自然浄化の基盤

→ 小型湿地システムと組むと、ほぼエネルギーゼロで水循環が回る。

② 鶏の平飼いの排水や堆肥液の前処理

→ 窒素肥料化を防ぎ、畑への投入を安定。

③ 果樹園・畑の栄養塩循環の要

→ “汚水”が“栄養水”に変わる。

④ 自立型エコビレッジの象徴技術

AIガバナンス化しやすい：

センサーで水質モニタリング

AIがバクテリアの活性を推定

メンテ時期を予測

水量・流速・層厚を最適化
→ 「自然浄化 × AIチューニング」という未来像に合う。

—

🧭 5. 注意点（課題）

立ち上がりに1〜3ヶ月かかる（微生物定着）

過負荷に弱い

固定した重金属が飽和すると交換が必要

設計をミスると水がショートカットして浄化しない

冬期に活性が落ちる（温暖な遠州なら影響小）

—

・コンポストトイレ
🧪 分離式 × 乾式のシステム構造
① 便は後方の堆肥チャンバーへ（固体）

木質素材（おがくず、籾殻など）を入れて水分を吸収

微生物が好気性で分解

排気パイプから空気を上向きに流す

年1回〜2回の取り出し

② 尿は前方スロットから別タンクへ（液体）

タンクで発酵 → 液肥に再利用可能
（水で5〜10倍希釈すれば果樹・森に最適）

タンクは満タンで運ぶだけなので軽い

③ 換気システム

自然通気でも可能

小型ファンを付けるとより理想的（1〜2W程度）

室内にニオイが戻らない構造が作れる

🌬 ニオイがほぼゼロになる理由

固体と尿を分離 → アンモニアの化学反応が起こらない

炭素材（おがくず）が水分を吸収 → 嫌気化しない

上方向への換気 → 室内に匂いが戻らない

微生物が分解 → 森の土の匂いに近づく

この方式は、適切に設計すれば「森林の腐葉土のような香り」になります。
